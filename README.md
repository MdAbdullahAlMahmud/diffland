# diffland

> Spatial git diff visualizer — see your codebase as a living village, not a wall of text.

**Status:** Phase 0 — foundations. Not yet usable. See [DIFFLAND_ARCHITECTURE.md](./DIFFLAND_ARCHITECTURE.md) for the plan.

## Dev setup

```bash
pnpm install
pnpm build
pnpm test
```

Requires Node 18+ and pnpm 9+.
